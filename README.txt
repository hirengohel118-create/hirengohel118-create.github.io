Vamika Estate CRM - PWA (basic starter)

How to use:
1. Upload the entire folder to GitHub repository and enable GitHub Pages (root).
2. Open the URL (your-username.github.io/repo) on phone and Add to Home Screen.
3. The app uses localStorage - data stays in browser (no server).
4. Use Call button to call; WhatsApp button opens chat with prefilled message.
5. Toggle dark/light using the top-right button.

Files:
- index.html
- styles.css
- app.js
- manifest.json
- sw.js
